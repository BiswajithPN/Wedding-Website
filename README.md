## 🌟 Overview

Welcome to the official wedding website for **Prabhu & Swathi**. This digital space serves as a beautiful gallery and archive of their special day, held on **December 1, 2025**, in the serene town of **Rameshwaram**.

Designed with a premium, minimalist aesthetic, this website allows friends and family to relive the magic of the wedding through high-quality photos, a detailed timeline of events, and a dedicated download section for cherished memories.

## ✨ Key Features

- **🏆 Premium Gallery**: A high-end, responsive photo gallery with a lightbox viewer for an immersive experience.
- **📥 Instant Downloads**: Fixed the download functionality! Users can now download individual high-res images directly from the gallery or full albums from the download center.
- **📅 Event Timeline**: A beautiful chronological journey of the wedding day, from traditional morning rituals to the evening reception.
- **📱 Responsive Design**: Fully optimized for mobile, tablet, and desktop viewing, ensuring a seamless experience on any device.
- **🎨 Elegant UI**: A sophisticated black and gold theme using *Playfair Display* and *Inter* typography for a luxurious feel.

## 🛠️ Tech Stack

This project is built using modern web standards to ensure speed, accessibility, and high performance.

- **Frontend**: Semantic HTML5, Vanilla CSS3 (Custom Properties), Modern JavaScript (ES6+).
- **Typography**: [Google Fonts](https://fonts.google.com/) (Playfair Display & Inter).
- **Icons**: Custom SVG icons for a crisp look on all resolutions.
- **Interactions**: Custom-built lightbox and mobile navigation for a lightweight, dependency-free experience.

## 📂 Project Structure

```bash
├── css/              # Custom styling with premium tokens
├── images/           # High-resolution wedding photography
├── js/               # Core logic and gallery interactions
├── index.html        # Home & Welcome screen
├── gallery.html      # The digital photo album
├── timeline.html     # The story of December 1st
├── downloads.html    # Photo download center
└── about.html        # About the couple
```

## 🚀 Getting Started

To view the website locally, follow these simple steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/BiswajithPN/Wedding-Website.git
   ```
2. **Navigate to the directory**:
   ```bash
   cd Wedding-Website
   ```
3. **Open `index.html`**:
   Simply open the `index.html` file in your preferred web browser.
